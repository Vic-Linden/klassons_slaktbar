# Bilder för Klassons Sportbar

För att hemsidan ska se komplett ut, lägg till följande bilder i denna mapp:

## Obligatoriska bilder:

1. **hero-bg.jpg** - Bakgrundsbild för hero-sektionen
   - Rekommenderad storlek: 1920x1080px
   - Förslag: Bild på baren inifrån med TV-skärmar och gäster

2. **guinness.jpg** - Bild för Meny & Dryck-kortet
   - Rekommenderad storlek: 800x800px
   - Förslag: Närbild på Guinness-öl eller dryckesmeny

3. **table.jpg** - Bild för Boka Bord-kortet
   - Rekommenderad storlek: 800x800px
   - Förslag: Bild på bordssättning i baren

4. **coworking.jpg** - Bild för Coworking Space-kortet
   - Rekommenderad storlek: 800x800px
   - Förslag: Bild på coworking-utrymmet med bord och stolar

## Bildformat:
- Format: JPG eller PNG
- Optimera bilderna för webben (komprimera för snabbare laddning)
- Se till att bilderna har god kvalitet men inte är för stora
